# OpenAPI Generator

## Configuration File

Users can include a `.config.json` or `.config.yaml` file to use an OpenAPI Generator configuration file. The configuration file must be placed right beside the OpenAPI specificaitons document and with the extension `.config.json` or `.config.yaml` depending on the format of the specifications file
