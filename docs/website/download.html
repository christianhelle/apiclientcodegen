Installation Requirements

For Visual Studio Extensions

System Requirements

  • Visual Studio 2017, 2019, or 2022 - Any edition (Community, Professional, Enterprise)
  • .NET Framework 4.7.2 or later - Usually included with Visual Studio
  • Windows 10 or later - For Visual Studio on Windows

For VS Code Extension

VS Code Requirements

  • Visual Studio Code - Latest stable version recommended
  • .NET SDK - Required for CLI tool execution
  • Cross-platform support - Windows, macOS, Linux

For Command Line Tool

CLI Tool Requirements

  • .NET 6.0 runtime or later - Core requirement
  • Java Runtime Environment - For Swagger Codegen CLI and OpenAPI Generator
  • NPM - For AutoRest and NSwag CLI installations
  • Internet access - For downloading dependencies on first use

Installation Tips and Best Practices

üí° Pro Tips for Smooth Installation

  • First-time use delay - Some code generators download on demand, expect initial delays
  • Java configuration - Configure Java path in settings if multiple installations exist
  • Network access - Ensure internet access for dependency downloads
  • Proxy settings - Configure NPM and Java tools for corporate proxies

Corporate Environment Setup

  1. Java Configuration - Set JAVA_HOME environment variable or configure in extension settings
  2. NPM Proxy Setup - Configure NPM for corporate proxies: npm config set proxy http://proxy:port
  3. Firewall Rules - Allow downloads from GitHub, NPM registry, and Maven central
  4. JAR File Caching - Pre-download JAR files to shared locations if needed

Getting Started After Installation

  1. Restart your IDE to ensure the extension is properly loaded
  2. Test with a sample API using the Swagger Petstore (https://petstore.swagger.io/v2/swagger.json)
  3. Configure your preferences in the Tools ‚Üí REST API Client Code Generator settings
  4. Add your first REST API Client to a project
  5. Explore the context menus for advanced features

Troubleshooting Installation Issues

Common Issues and Solutions

Extension Not Loading

  • Restart Visual Studio completely
  • Check Extensions ‚Üí Manage Extensions ‚Üí Installed
  • Ensure .NET Framework requirements are met

Java Not Found Errors

  • Install Java JDK 8 or later
  • Set JAVA_HOME environment variable
  • Configure Java path in extension settings

NPM Installation Issues

  • Install Node.js and NPM
  • Configure proxy settings if behind corporate firewall
  • Clear NPM cache: npm cache clean --force
