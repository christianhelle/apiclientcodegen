## Installation Tips and Best Practices

<div class="feature-highlight">
  <h3>üí° Pro Tips for Smooth Installation</h3>
  <ul>
    <li><strong>First-time use delay</strong> - Some code generators download on demand, expect initial delays</li>
    <li><strong>Java configuration</strong> - Configure Java path in settings if multiple installations exist</li>
    <li><strong>Network access</strong> - Ensure internet access for dependency downloads</li>
    <li><strong>Proxy settings</strong> - Configure NPM and Java tools for corporate proxies</li>
  </ul>
</div>

### Corporate Environment Setup

<ol class="installation-steps">
  <li><strong>Java Configuration</strong> - Set JAVA_HOME environment variable or configure in extension settings</li>
  <li><strong>NPM Proxy Setup</strong> - Configure NPM for corporate proxies: <code>npm config set proxy http://proxy:port</code></li>
  <li><strong>Firewall Rules</strong> - Allow downloads from GitHub, NPM registry, and Maven central</li>
  <li><strong>JAR File Caching</strong> - Pre-download JAR files to shared locations if needed</li>
</ol>

## Troubleshooting Installation Issues

### Common Issues and Solutions

<div class="config-section">
  <h4>Extension Not Loading</h4>
  <ul>
    <li>Restart Visual Studio completely</li>
    <li>Check Extensions ‚Üí Manage Extensions ‚Üí Installed</li>
    <li>Ensure .NET Framework requirements are met</li>
  </ul>
</div>

<div class="config-section">
  <h4>Java Not Found Errors</h4>
  <ul>
    <li>Install Java JDK 8 or later</li>
    <li>Set JAVA_HOME environment variable</li>
    <li>Configure Java path in extension settings</li>
  </ul>
</div>

<div class="config-section">
  <h4>NPM Installation Issues</h4>
  <ul>
    <li>Install Node.js and NPM</li>
    <li>Configure proxy settings if behind corporate firewall</li>
    <li>Clear NPM cache: <code>npm cache clean --force</code></li>
  </ul>
</div>
