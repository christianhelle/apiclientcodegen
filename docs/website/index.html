Supported Code Generators

NSwag

Generates C# REST API clients using the NSwag.CodeGeneration.CSharp package. Produces clean, strongly-typed code with full async support.

OpenAPI Generator

Uses OpenAPI Generator v7.14.0 to create comprehensive client libraries with extensive customization options and multiple output formats.

Microsoft Kiota

Microsoft's modern API client generator with support for Microsoft Graph and other APIs, featuring strong typing and authentication.

Refitter

Generates Refit interfaces for reactive REST clients with RxJava support and comprehensive testing capabilities.

Swagger Codegen CLI

Classic Swagger code generation with proven reliability and extensive language support for legacy and modern projects.

AutoRest

Microsoft's AutoRest generator for Azure services and general REST APIs with advanced authentication and retry policies.

Quick Start

Get started with REST API Client Code Generator in just a few steps:

  1. Install the extension from the Visual Studio Marketplace or download the VSIX file
  2. Add a new REST API Client to your project from the Add New Item dialog
  3. Enter your OpenAPI specification URL (e.g., https://petstore.swagger.io/v2/swagger.json)
  4. Choose your preferred code generator and configure the namespace
  5. Generate your client code and start making API calls!

Command Line Tool

For automation and CI/CD scenarios, use our cross-platform command line tool:

# Install the global tool
dotnet tool install --global rapicgen

# Generate C# client using NSwag
rapicgen csharp nswag swagger.json MyNamespace ./MyClient.cs

# Generate TypeScript client
rapicgen typescript Angular swagger.json ./typescript-client

# Generate JMeter test plans
rapicgen jmeter swagger.json
